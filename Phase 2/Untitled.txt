## B2. Create a Linked Server (to YOUR PC)

Replace `YOUR_PC_IP` with your real IP
Replace `YOUR_SERVER_NAME` if needed (often `.` or `DESKTOP-XXX`)

```sql
EXEC sp_addlinkedserver
    @server = 'CourseMgmt_Link',
    @srvproduct = '',
    @provider = 'SQLNCLI',
    @datasrc = 'YOUR_PC_IP';
GO
```

ðŸ“Œ Example:

```sql
@datasrc = '192.168.1.10'
```

## B3. Configure login mapping (THIS IS CRITICAL)

```sql
EXEC sp_addlinkedsrvlogin
    @rmtsrvname = 'CourseMgmt_Link',
    @useself = 'FALSE',
    @rmtuser = 'lms_remote_login',
    @rmtpassword = 'Strong@123';
GO
```

ðŸ“Œ This tells SQL Server how to authenticate to your PC.
